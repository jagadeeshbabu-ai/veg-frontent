🥕 Vegetable Store Frontend

A modern and responsive Vegetable Store frontend application built using React.js, Vite, and Tailwind CSS. This project provides an intuitive shopping experience where users can browse fresh vegetables, view product details, and manage their cart efficiently.

📌 Project Overview

The Vegetable Store Frontend is designed to simulate an online grocery shopping experience. Users can explore a variety of vegetables, view pricing information, and add items to their cart through a clean and user-friendly interface.

This project focuses on frontend development concepts such as component-based architecture, state management, responsive design, and modern UI development practices.

🚀 Features
🥦 Browse a variety of fresh vegetables
🔍 Search vegetables by name
🛒 Add products to cart
➕ Increase or decrease cart quantity
❌ Remove items from cart
💰 View total cart price
📱 Fully responsive design
🎨 Modern and attractive user interface
⚡ Fast performance with Vite
